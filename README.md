Prerequisite
-----
* Nodejs >= v12
* Yarn
* ProgresSql >= v13.2

Building and test
-----

* Build
  
```bash
$ yarn install --frozen-lockfile
```

Run
-----

* For development

```bash
$ yarn start
```

* For production

```bash
$ yarn run prod
```
